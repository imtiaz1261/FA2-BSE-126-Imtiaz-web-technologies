Lecture 4: Tables, Div, Span, and Layout Design

Introduction to HTML Layout

HTML provides basic elements for structuring web content. The focus of this lecture is on tables, div, and span elements for layout design.

  • These elements help organize content within a webpage.
  • <table> organizes data into rows and columns.
  • <div> groups content for structure and styling.
  • <span> styles inline portions of text.

Example of div & span:

<div> This is a block of text. <span style="color:red;">Important</span> </div>

What Are HTML Tables?

HTML tables are used to present data in a tabular format consisting of rows and columns.

  • Commonly used for structuring and displaying data.
  • Each table consists of <tr> (table row), <th> (header), and <td> (data cell) elements.
  • In the past, tables were used for page layout — but this practice is now deprecated.

Example:

<table border="1"> <tr> <th>Header 1</th> <th>Header 2</th> </tr> <tr> <td>Data 1</td> <td>Data 2</td> </tr> </table>

Basic Table Structure

A simple HTML table includes table, row, and cell elements. It organizes data into readable sections.

  • <table> : Defines the table.
  • <tr> : Defines a row.
  • <th> : Table header (bold and centered by default).
  • <td> : Table data (cell).

Example:

<table> <thead> <tr> <th>Header 1</th> <th>Header 2</th> </tr> </thead> <tbody> <tr> <td>Data 1</td> <td>Data 2</td> </tr> </tbody> </table>

This is an example of a simple table with headers and data rows.
