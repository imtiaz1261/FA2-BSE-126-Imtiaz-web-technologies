Lecture 8: CSS Selectors

Descendant and Child Selectors

Descendant selectors target all nested elements inside a parent (child + grandchild). Child selectors only target the direct children of an element.

Example HTML:

    
  
  1. List item one.
    2. 
  
  2. List item two.
    3. 
  
      
    
    • Sublist item a.
      • 
    
    • Sublist item b.
      • 
  
    
  
  3. List item three.
    4.

Example CSS:

ol > li {
  color: blue; /* child selector */
}

ol li {
  color: blue; /* descendant selector */
}

li {
  color: blue;
}

Attribute Selectors

Attribute selectors style elements based on the presence or value of attributes.

Example HTML:

    
  
  • Google
    • 
  
  • Facebook
    • 
  
  • Amazon
    •

Example CSS:

a {
  color: green;
}

a[href="https://facebook.com"] {
  color: blue;
}

In this example, all links are green, but the one pointing to Facebook becomes blue.

Pseudo-Class Selectors

Pseudo-classes define a special state of an element — for example, when a user hovers or focuses on it.

Example HTML:

Example CSS:

button {
  background-color: green;
  font-size: 24px;
  width: 200px;
  height: 50px;
}

button:hover {
  background-color: orange;
}

Here, the button changes its color to orange when the user hovers over it.
