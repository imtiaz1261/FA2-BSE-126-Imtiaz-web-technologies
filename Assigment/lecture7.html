Lecture 7: CSS

CSS Introduction

CSS (Cascading Style Sheets) is the language used to style and layout web pages.

  • CSS describes how HTML elements are displayed on screen, paper, or other media.
  • It helps in separating content from design.
  • CSS saves time — one stylesheet can control multiple web pages.
  • External stylesheets are stored in .css files.

CSS Syntax

A CSS rule consists of a selector and a declaration block:

selector { property: value; }

Example:

h1 { color: red; text-align: right; } p { font-family: Arial; font-size: 20px; }

Common CSS Properties

  • color – sets the text color
  • text-align – aligns the text (left, right, center)
  • font-family, font-size, font-weight – control font style and appearance
  • width, height – define element dimensions
  • margin, padding – create space around and inside elements
  • border – defines the border around elements

Example:

div { color: blue; width: 200px; height: 200px; border: 3px dashed black; margin: 20px; padding: 20px; }

How to Add CSS

There are three ways to insert CSS into an HTML document:

  • External CSS – use a separate .css file linked with the HTML file.
  • Internal CSS – include CSS inside a <style> tag in the HTML <head>.
  • Inline CSS – write CSS directly inside an element’s style attribute.

Example: Inline CSS

<h1 style="color: blue; text-align: center;">Hello</h1> <body style="color: blue; text-align: center;">

Example: External CSS

h1 { color: blue; text-align: left; font-family: Arial, sans-serif; font-size: 28px; font-weight: normal; }
