Lecture 5: Introduction to HTML5

What is HTML5?

HTML5 is the latest version of the HTML standard. It introduces new elements, attributes, and behaviors that enhance modern web development.

  • Focuses on better semantics, multimedia support, and device independence.
  • Provides structure and features for responsive and interactive web applications.

Doctype Declaration:

<!DOCTYPE html>

Key Features of HTML5

  • Responsive Design: Websites automatically adjust layout and content based on screen size.
  • Viewport Meta Tag: Ensures proper scaling across different devices.
  • Flexible Multimedia: Supports native <audio> and <video> without third-party plugins like Flash.
  • Cross-Platform Compatibility: Includes the <canvas> element for graphics and interactive content across devices.
  • Forms and Input Types: Adds new types like email, tel, and date for better device adaptation.
  • Offline Support: Technologies such as localStorage and service workers enable offline functionality.

HTML5 vs Previous Versions

HTML4: Focused on formatting using elements like <div> and <span>.

HTML5: Introduced semantic tags to improve readability, accessibility, and structure.

The Importance of Semantic HTML

  • Improves accessibility, SEO, and maintainability.
  • Helps browsers and search engines understand webpage content.

Examples of Semantic Elements

  • <header>, <footer>, <article>, <section>, <nav>, <aside>

Example:

<article> <h2>Latest News</h2> <p>This is a news article.</p> </article>
