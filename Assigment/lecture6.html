Lecture 6: HTML Forms

Introduction to HTML Forms

HTML Forms provide a way to collect user input in web applications. Forms are a core part of interactive websites such as login pages, registration forms, and feedback systems.

  • Used for data submission from the user to the server.
  • Help gather information such as text, emails, passwords, or selections.
  • Communicate with web servers to process or store the submitted data.

The
Element

The <form> tag defines a container for collecting user input. It can include multiple elements such as text fields, buttons, and dropdowns.

Example:

<form action="/submit" method="post"> <!-- Form content here --> </form>

Common form elements include:

  • <label> – Describes an input field.
  • <input> – Collects user data.
  • <select> and <option> – Create dropdown lists.
  • <textarea> – For multi-line text input.
  • <button> – For submitting or resetting the form.

Form Attributes

  • action: Specifies the URL where the form data will be sent.
  • method: Defines how the data is sent (GET or POST).

GET vs POST

  • GET: Sends data through the URL (used for simple queries).
  • POST: Sends data in the request body (used for secure or large data).

Example:

<form action="/submit" method="post"> <label for="name">Name:</label> <input type="text" id="name" name="name"> <button type="submit">Submit</button> </form>

Other important form attributes include name, value, placeholder, and required.
