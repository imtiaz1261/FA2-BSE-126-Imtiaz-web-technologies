Lecture 3: Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and design web pages.

  • Defines the structure of a webpage using tags and elements.
  • Forms the foundation of all web content.
  • Works together with CSS (for styling) and JavaScript (for interactivity).

Example:

<h1>Hello, World!</h1>

History of HTML

HTML was developed by Tim Berners-Lee in 1991 as a simple way to share and format documents on the World Wide Web.

  • 1991: HTML 1.0 introduced basic tags for documents.
  • 1995: HTML 2.0 standardized, improving browser compatibility.
  • 1997–1999: HTML 3.2 and 4.0 added support for multimedia, scripting, and styling.
  • 2014: HTML5 released with powerful features for modern web applications.

Evolution of HTML

Over the years, HTML has evolved to support dynamic, responsive, and media-rich experiences on the web.

  • HTML 1.0: Basic structure and hyperlinks.
  • HTML 2.0–4.0: Introduced forms, tables, and style integration.
  • XHTML: Brought XML-based syntax for stricter coding standards.
  • HTML 5: Added semantic elements, audio/video tags, and APIs for offline and interactive features.

Today, HTML5 forms the backbone of nearly every website and web application worldwide.
